# Python-Base
人工智能Python基础编程课程
## 课程概览
欢迎加入Python基础课程！本课程由Momodel提供，旨在帮助初学者掌握Python编程语言的基础知识和技能。     
通过本课程，您将学习到基础数据类型、字符串操作、列表和元组等核心概念，并通过理论介绍和代码实操，掌握基本的Python语法和流程控制。  
## 课程内容
  第一课	Python 基础数据类型   
  第二课	Python 字符串及其操作   
  第三课	Python 列表和元组   
  第四课	Python 字典和集合  
  第五课	Python 流程控制
